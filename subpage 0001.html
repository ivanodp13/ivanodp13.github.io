Conversor de Unidades

APP ANDROID
Previous Next

Esta app se ha realizado íntegramente con el programa Android Studio con lenguaje Java y xml para la interfaz. La aplicación es un conversor de unidades, el cual nos permitirá saber de forma inmediata la equivalencia de un valor de una unidad a otra

La app nos permite elegir entre las siguientes magnitudes: Temperatura, divisas, datos, longitud, masa, volumen, unidades astronómicas, presión, velocidad y área. Cada una de estas unidades tiene disponibles sus unidades principales además de las del sistema internacional.
