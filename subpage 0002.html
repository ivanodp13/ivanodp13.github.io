Juego Memory Trainer

APP iOS
Previous Next

Esta app se ha realizado íntegramente con el programa Xcode de MacOS con lenguaje Swift. La aplicación es un juego para entrenar la memoria del usuario.

La app consta de 2 vistas. La primera nos muestra las instrucciones de uso y un slider para cambiar el nivel de dificultad, además de un botón para iniciar la partida. Una vez pulsemos el botón de inicio se ocultarán los elementos anteriores y se mostrará una secuencia de imágenes que debemos memorizar, la velocidad de estas dependerá de la dificultad que hayamos elegido anteriormente.

Tras la secuencia de imágenes aparecerá un marcador con los puntos que vayamos logrando y en la parte inferior los botones haciendo referencia a las imágenes anteriores.

Según vayamos acertando o no el orden de las imágenes iremos sumando puntos. Además, si se aciertan 2 o más seguidas, se sumará un punto extra, esto se denominará racha.

La app ha sido desarrollada únicamente para iOS y está optimizada para todos los modelos de iPhone desde el SE hasta el 11 Pro Max y para iPad desde el modelo de 7,9″ hasta el Pro de 12,9″.
